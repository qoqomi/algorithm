## 2-1. 어레이 링크드리스트 
> 💡 어느 상황에 사용하는지를 정확히 알고 있을 것 

Array 
- 특정 원소 조회 : O(1)
- 중간에 삽입: O(N)
- 데이터 추가: 데이터 추가 시 모든 공간이 다 차버렸다면 새로운 메모리 공간을 할당 받아야 한다. 
- 정리: 데이터에 접근하는 경우가 빈번하다면 Array 사용

링크드리스트 
- 특정 원소 조회 : O(N)
- 중간에 삽입: O(1)
- 데이터 추가: 모든 공간이 다 찼어도 동적 노드만 생성하면 된다. 
- 정리: 삽입과 삭제가 빈번하게 일어난다면, 링크드리스트를 사용
> 연결고리: 포인터, 화물칸: 노드

## 2-2. 클래스 

클래스: 클래스는 객체를 정의하기 위한 설계도 또는 틀이다. 
객체:
클래스 > 객체1, 객체2

```python
# Person 클래스의 객체로 주소에 위치해 있다.
person1 = Person() # 하나의 객체를 생성하겠다고 정의 
person2 = Person()
```

생성자: 객체를 만들 때 만드는 시점에서 호출되는 함수 
클래스 내부 함수는 Method라고 부른다.
```python

class Person:
    def __init__(self,name):
        self.name = name # named이라는 주소에 할당
        print(self)

person_1 = Person("유재석")    
person_2 = Person("박명수")
```
